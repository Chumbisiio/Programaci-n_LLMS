El Problema:

En un torneo de videojuegos multijugador se registran las estadísticas de cada jugador
en un DataFrame. Sin embargo, el DataFrame puede contener filas con valores nulos (NaN)
en cualquiera de sus columnas numéricas, los cuales deben ser descartados antes del análisis.

Tu Misión:

Escribe una función llamada limpiar_y_rankear_jugadores(df, top_n) que realice lo siguiente:

- Reciba un DataFrame df con las columnas: 'jugador' (str), 'kills' (int/float),
  'deaths' (int/float), 'assists' (int/float), 'partidas_jugadas' (int/float).
- Elimine todas las filas que contengan al menos un valor NaN en cualquier columna.
- Calcule para cada jugador una métrica 'kda' usando la fórmula:
    kda = (kills + assists) / deaths  cuando deaths > 0
    kda = (kills + assists)           cuando deaths == 0
  (es decir, usa deaths con un mínimo de 1 para evitar división por cero).
- Devuelva un DataFrame con las columnas ['jugador', 'kda', 'partidas_jugadas'],
  con exactamente las top_n filas de mayor kda (ordenadas de mayor a menor),
  con el índice reiniciado (reset_index con drop=True).

Restricciones:
- Usar solo pandas y numpy (no sklearn).
- No usar loops explícitos (for/while) para los cálculos.
- Si hay menos filas válidas que top_n, devolver todas las filas válidas.

Ejemplo:

    import pandas as pd
    import numpy as np

    df = pd.DataFrame({
        'jugador':          ['Alice', 'Bob',  'Carol', 'Dave', 'Eve'],
        'kills':            [20,       5,      15,      None,   10],
        'deaths':           [4,        10,     3,       8,      0],
        'assists':          [8,        2,      12,      5,      6],
        'partidas_jugadas': [10,       8,      12,      7,      9]
    })

    resultado = limpiar_y_rankear_jugadores(df, top_n=3)
    print(resultado)
    #   jugador        kda  partidas_jugadas
    # 0   Carol   9.000000              12.0
    # 1    Eve   16.000000               9.0   <- deaths=0, kda=kills+assists=16
    # 2   Alice   7.000000              10.0
    # (orden real depende de los valores calculados)

El Problema:

Una plataforma de matchmaking de videojuegos quiere segmentar automáticamente a sus
jugadores en grupos de habilidad similares (por ejemplo: principiante, intermedio, experto)
utilizando sus estadísticas de juego. El dataset puede contener valores nulos que deben
imputarse antes del clustering.

Tu Misión:

Escribe una función llamada agrupar_jugadores_por_habilidad(X, n_clusters) que:

- Reciba:
    - X (np.ndarray): matriz de estadísticas de jugadores de forma (n_samples, n_features).
      Puede contener valores NaN.
    - n_clusters (int): número de grupos de habilidad a formar.
- Impute los valores faltantes en X usando la media de cada columna
  (SimpleImputer con strategy='mean' de sklearn).
- Escale las características para que tengan media 0 y desviación estándar 1
  (StandardScaler de sklearn).
- Aplique KMeans con n_clusters=n_clusters, random_state=42 y n_init=10.
- Devuelva un array de numpy de forma (n_samples,) con las etiquetas de cluster
  asignadas a cada jugador.

Restricciones:
- Librerías permitidas: numpy, sklearn.
- No usar pandas.
- El random_state del KMeans SIEMPRE debe ser 42 y n_init=10.

Ejemplo:

    import numpy as np

    # Simula estadísticas: [kda, win_rate, horas_jugadas, nivel_cuenta]
    X = np.array([
        [3.5,  0.65, 500,  30],
        [0.8,  0.40, 50,    5],
        [np.nan, 0.55, 200, 15],
        [5.1,  0.72, 1200, 80],
        [0.5,  0.35,  20,   2],
        [4.8,  0.70, 950,  75]
    ])

    etiquetas = agrupar_jugadores_por_habilidad(X, n_clusters=3)
    print(etiquetas)         # e.g. array([1, 2, 2, 0, 2, 0])
    print(etiquetas.shape)   # (6,)
    # Los valores exactos dependen de KMeans; lo importante es que sean enteros
    # entre 0 y n_clusters-1 y que jugadores similares queden en el mismo grupo.
